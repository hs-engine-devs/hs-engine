[h1]
Adding songs to your mod
[/h1]

[text]
To add custom songs to your mod in the HS Engine, follow the folder structure and file naming conventions carefully. This will ensure the engine properly loads and plays your songs.
[/text]

[h1]
Folder structure
[/h1]

[text]
Your mod folder should be organized like this:
[/text]

[code]
'mods/'
  'your-mod/'
    'songs/'
      'your-song/'
        'Inst.ogg'        // - Instrumental audio track
        'Voices.ogg'      // - Vocals audio track (optional)
    'data/'
      'charts/'
        'your-song/'
          'your-song.json'      // - Chart data file (optional)
          'your-song-hard.json'
          'your-song-easy.json'
[/code]

[text]
- Inst.ogg and Voices.ogg are the sound files used by the game.
- Chart files describe the notes and timings for the song.
- Chart file names can include optional difficulty suffixes like -easy or -hard.
[/text]

[h1]
Example
[/h1]

[text]
If you want to add a song named "CoolBeat" to your mod named "my_mod", your files should look like this:
[/text]

[code]
'mods/my_mod/songs/CoolBeat/Inst.ogg'
'mods/my_mod/songs/CoolBeat/Voices.ogg'
'mods/my_mod/data/charts/CoolBeat/CoolBeat.json'
'mods/my_mod/data/charts/CoolBeat/CoolBeat-hard.json'
[/code]

[h1]
Song metadata
[/h1]

[text]
To add songs to the freeplay menu and your mod’s song list, you need to create a file named songList.txt inside your mod folder (e.g. mods/your-mod/songList.txt) with the following format:
[/text]

[code]
SongName:Character:WeekNumber:Difficulty1,Difficulty2,Difficulty3
[/code]

[text]
- **SongName**: The folder name of your song.
- **Character**: The character associated with the song.
- **WeekNumber**: The week number this song belongs to.
- **Difficulties**: Comma-separated difficulties (e.g. easy,normal,hard).
[/text]

[text]
Example:
[/text]

[code]
CoolBeat:bf:0:easy,normal,hard
[/text]

[h1]
Loading songs in the game
[/h1]

[text]
The game automatically loads songs from enabled mods inside the mods/ folder.
Make sure your audio files are in songs/your-song/ and charts in data/charts/your-song/.
[/text]

[text]

Follow these steps to add your custom songs successfully!
[/text]

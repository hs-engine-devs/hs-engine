[h1]
How to add custom characters
[/h1]

[text]
To add custom characters in the game mod, you can create your character data in either JSON or TXT format.
Both formats are supported and you can choose the one you prefer.

The character files should be placed in the folder:
mods/your-mod/data/characters/
[/text]

[h1]
TXT Format Example
[/h1]

[text]
The TXT format is a simple text file where you define properties and animations line by line.
[/text]

[code]
lang=txt
spritePath=characters/bf
healthIcon=bf
scale=1
flipX=false
antialiasing=true
singDuration=4
cameraOffset=0:0
characterOffset=0:0
animation=idle, BF idle dance, 24, true, 0:0
animation=singLEFT, BF LEFT NOTE, 24, false, -10:0
animation=singDOWN, BF DOWN NOTE, 24, false, 0:-10
animation=singUP, BF UP NOTE, 24, false, 0:10
animation=singRIGHT, BF RIGHT NOTE, 24, false, 10:0
[/code]

[h1]
JSON Format Example
[/h1]

[text]
Alternatively, you can create a JSON file with the same data structured as key-value pairs and arrays.
[/text]

[code]
{
  "spritePath": "characters/bf",
  "healthIcon": "bf",
  "scale": 1,
  "flipX": false,
  "antialiasing": true,
  "singDuration": 4,
  "cameraOffset": [0, 0],
  "characterOffset": [0, 0],
  "animations": [
    { "anim": "idle", "name": "BF idle dance", "fps": 24, "loop": true, "offset": [0, 0] },
    { "anim": "singLEFT", "name": "BF LEFT NOTE", "fps": 24, "loop": false, "offset": [-10, 0] },
    { "anim": "singDOWN", "name": "BF DOWN NOTE", "fps": 24, "loop": false, "offset": [0, -10] },
    { "anim": "singUP", "name": "BF UP NOTE", "fps": 24, "loop": false, "offset": [0, 10] },
    { "anim": "singRIGHT", "name": "BF RIGHT NOTE", "fps": 24, "loop": false, "offset": [10, 0] }
  ]
}
[/code]

[text]
Make sure to save your files as `.txt` or `.json` respectively and place them in the correct folder.

There is also a Character Editor tool available that can help you create and manage these files easily. This tool will be described in other chapters.
[/text]

[h1]
Summary
[/h1]

[text]
- You can use TXT or JSON format for character data.
- Both formats support the same properties and animations.
- Place your files inside mods/your-mod/data/characters/.
- Use the Character Editor for easier editing (covered in other chapters).
- Load your characters in-game by referencing their folder names.

Following these formats ensures your custom characters load correctly.
[/text]
